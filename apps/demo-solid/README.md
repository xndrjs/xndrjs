# @xndrjs/demo-solid

Solid demo that replicates the UI of `demo-react` using `@xndrjs/adapter-solid`.

## Commands

- `pnpm dev` - starts Vite
- `pnpm build` - tsc + vite build
- `pnpm preview`
- `pnpm lint`

## Notes
- Title: "xndr Demo - Solid" (only visual difference compared to demo-react).
- Uses `@xndrjs/adapter-solid` to connect StatePort/ComputedValue to Solid.
